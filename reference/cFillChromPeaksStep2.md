# Intensive peak extraction step performed file-wise (parallel in cluster workflows)

Intensive peak extraction step performed file-wise (parallel in cluster
workflows)

## Usage

``` r
cFillChromPeaksStep2(input, output)
```

## Arguments

- input:

  path to the prepared data file from cFillChromPeaksStep1

- output:

  the output file path for storage of extracted peaks

## Value

NULL (writes output to file)

# fill chrom peak function 3

fill chrom peak function 3

## Usage

``` r
cFillChromPeaksStep3(inputFromStep1, inputFromStep2, output, param, settings)
```

## Arguments

- inputFromStep1:

  Output of cFillChromPeaksStep1() function

- inputFromStep2:

  Output of cFillChromPeaksStep2() function

- output:

  the output which contains the result of the preprocessing

- param:

  FillChromPeaksParam object

- settings:

  settings list from settings.yaml

## Value

a preprocessed xcms object
